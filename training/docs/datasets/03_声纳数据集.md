## 可实现的深度学习算法

### 1. 声纳图像分类 (Sonar Image Classification)

声纳图像与自然图像差异大（低分辨率、强噪声、纹理少），需要专门的分类策略。

| 模型 | 适用场景 | 备注 |
|------|----------|------|
| **ResNet-18/50** | 中等规模基准 | 预训练权重可加速收敛 |
| **EfficientNet-B0~B3** | 轻量级高效分类 | 适合移动端/实时推理 |
| **Vision Transformer (ViT)** | 全局特征提取 | 声纳图全局结构信息 |
| **SonarNet** (自设计) | 声纳特有特征 | 小卷积核 + 频域增强 |
| **Wavelet-CNN** | 频域+空域融合 | 小波变换提取声纳纹理 |

### 2. 声纳目标检测 (Sonar Object Detection)

| 模型 | 适用场景 | 备注 |
|------|----------|------|
| **YOLOv5s/v8n** | 实时声纳检测 | 轻量骨干适合小目标 |
| **Faster R-CNN + FPN** | 高精度检测 | FPN 增强多尺度特征 |
| **CenterNet** | Anchor-free 检测 | 避免 anchor 设计 |

### 3. 小样本/不平衡学习 (Few-shot & Imbalanced Learning)

| 方向 | 方法 |
|------|------|
| 数据增强 | 声纳特有增强（斑点噪声、混响模拟、增益调整） |
| 少样本学习 | Prototypical Networks, Matching Networks, MAML |
| 类别重平衡 | Focal Loss, Class-Balanced Loss, LDAM Loss |
| 过采样 | SMOTE + 声纳特征空间插值 |

### 4. 声纳图像增强与超分辨 (Sonar Enhancement & Super-Resolution)

| 模型 | 说明 |
|------|------|
| **SRCNN / ESRGAN** | 声纳图像超分辨率重建 |
| **U-Net 去噪** | 声纳斑点噪声抑制 |
| **CycleGAN** | 仿真声纳 → 真实声纳风格迁移 |

### 5. 跨域声纳识别 (Cross-Domain Sonar Recognition)

| 方法 | 说明 |
|------|------|
| **DANN (Domain Adversarial NN)** | 不同声纳设备间的域自适应 |
| **MMD + Coral** | 分布对齐跨域分类 |
| **Fine-tuning** | 源域预训练 + 目标域微调 |

### 6. 声纳图像分割 (Sonar Image Segmentation)

| 模型 | 说明 |
|------|------|
| **U-Net / Attention U-Net** | 声纳目标前景分割 |
| **DeepLabV3+** | 空洞卷积多尺度分割 |
| **SAM (Segment Anything)** | 零样本声纳分割（需微调） |

## 训练建议

1. **预处理**: 声纳图像建议先做归一化（均值/标准差）、直方图均衡化增强对比度
2. **输入尺寸**: FLSMDD 和 NKSID 图像尺寸不统一，建议 resize 到 224×224 或 256×256
3. **数据划分**: 建议 70% 训练 / 15% 验证 / 15% 测试，按类别分层采样
4. **评估指标**: Accuracy, F1-Score, Confusion Matrix (类别不平衡时重点看 per-class F1)

# 多模态数据集 (Multimodal Dataset)

> 路径: `F:\716数据集\多模态数据集`
> 总文件数: **378,860**
> 模态: 可见光图像 + 4D 毫米波雷达 + AIS + XML 标注

---

## 数据集概览

包含两个大规模海事多模态感知数据集，支持目标检测、语义分割、水线分割、多传感器融合等任务。

### 数据目录结构

```
WaterScenes-Published/
├── images/          (54,120 张) — 可见光 RGB 图像
├── radar/           (108,241 个) — 4D 毫米波雷达点云帧
├── detection/       (108,240 个) — 目标检测标注 (XML 格式)
├── semantic/        (54,120 个) — 语义分割标注
├── waterline/       (54,120 个) — 水线/可航行区域分割标注
├── train.txt        — 训练集文件列表
├── val.txt          — 验证集文件列表
├── test.txt         — 测试集文件列表
├── information_list.csv — 场景元信息 (光照/天气/水道/位置)
├── calib.zip        — 相机-雷达标定参数
├── gps.zip          — GPS 位置数据
├── imu.zip          — IMU 惯性测量数据
└── instance.zip     — 实例分割标注
```

### 可实现的深度学习算法

#### 1. 多模态 3D 目标检测

| 模型 | 输入模态 | 说明 |
|------|----------|------|
| **MVX-Net** | 图像 + 雷达 | 多视角融合 3D 检测 |
| **BEVFusion** | 相机 + Radar | 统一鸟瞰图感知 |
| **CRAFT** | 图像 + 雷达 | 跨模态时空融合 |
| **TransFuser** | 图像 + Radar | Transformer 多模态融合 |

#### 2. 语义分割 (Semantic Segmentation)

| 模型 | 说明 |
|------|------|
| **SegFormer** | Transformer 语义分割 |
| **DeepLabV3+** | 空洞卷积多尺度分割 |
| **PIDNet** | 实时双向语义分割 |

#### 3. 水线/可航行区域分割 (Waterline Segmentation)

| 模型 | 说明 |
|------|------|
| **U-Net / UNet++** | 经典分割基线 |
| **WaSR** | 专为水面障碍物分割设计 |
| **STDC-Seg** | 轻量实时分割 |

#### 4. 多任务学习 (Multi-Task Learning)

| 模型 | 联合任务 |
|------|----------|
| **YOLOP / YOLOPv2** | 检测 + 分割 + 车道线 |
| **Multi-Task Transformer** | 检测 + 分割 + 水线联合训练 |
| **HydraNet** | 多任务共享骨干网络 |

#### 5. 恶劣环境鲁棒感知

| 方法 | 说明 |
|------|------|
| **域泛化 (DG)** | 晴天→雨雾天的零样本泛化 |
| **域自适应 (DA)** | 少量目标域数据自适应 |
| **图像增强+去雨去雾** | 预处理提升恶劣天气图像质量 |

#### 6. 4D 雷达点云处理

| 模型 | 说明 |
|------|------|
| **PointPillars** | 快速柱体编码 3D 检测 |
| **RadarNet** | 毫米波雷达专有特征提取 |
| **RPFA-Net** | 雷达点云特征聚合 |

## 子集二：FVessel

| 属性 | 值 |
|------|-----|
| 路径 | `FVessel/` |
| 数据文件 | `01_Video+AIS.rar`, `02_Image+xml.rar` |
| 模态 | 视频 + AIS 轨迹 + 图像 + XML 标注 |

### 数据描述

FVessel 是一个融合船舶视频、AIS 数据和图像标注的综合数据集：

- **Video+AIS**: 视频帧与 AIS 轨迹的时间同步数据，可用于船舶检测-跟踪-AIS 关联
- **Image+xml**: 标注的船舶图像，可用于目标检测和分类

### 可实现的深度学习算法

| 算法方向 | 推荐模型 | 说明 |
|----------|----------|------|
| 视频目标检测 | YOLO + DeepSORT | 视频流舰船检测与跟踪 |
| 视觉-AIS 关联 | Cross-Modal Attention | 将视觉检测与 AIS 轨迹匹配 |
| 多目标跟踪 | ByteTrack | 视频中多船持续跟踪 |
| 船舶细粒度识别 | ResNet + Metric Learning | 基于图像的船舶型号识别 |

## 训练建议

1. **GPU 内存**: WaterScenes 全量训练建议 ≥24GB VRAM (RTX 3090/4090)
2. **数据加载**: 建议使用 `lmdb` 或 `webdataset` 格式加速 I/O
3. **多模态对齐**: 务必使用正确的标定参数 (`calib.zip`) 做图像-雷达空间对齐
4. **训练策略**: 建议先在单模态预训练，再联合多模态微调
5. **评估指标**: 目标检测用 mAP@0.5:0.95；分割用 mIoU；多模态额外关注恶劣环境的掉点

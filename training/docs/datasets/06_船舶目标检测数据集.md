# 船舶目标检测数据集 (Ship Detection Dataset)

> 路径: `F:\716数据集\船舶目标检测数据集`
> 总文件数: ~21,888
> 模态: 可见光图像

---

## 数据集概览

汇聚多个公开船舶检测数据集和基准，以 **YOLOv5 格式** 为核心组织，覆盖不同海域、不同尺度的船舶目标。

## 可实现的深度学习算法

### 1. 船舶目标检测 (Ship Detection)

| 模型 | 适用场景 | 说明 |
|------|----------|------|
| **YOLOv5s/m/l** | 通用船舶检测 | 训练速度快，工程成熟 |
| **YOLOv8n/s/m** | 最新 YOLO 系列 | 更强的检测性能 |
| **YOLOv10** | 端到端无 NMS | 推理更快 |
| **RT-DETR** | 实时 Transformer 检测 | 端到端，无需 NMS |
| **DETR / Deformable DETR** | 高精度检测 | Transformer 目标检测 |
| **Faster R-CNN** | 高精度需求 | 两阶段检测器 |

### 2. 小目标船舶检测 (Small Ship Detection)

远距离/低分辨率下的船舶检测是海事感知的难点。

| 方法 | 说明 |
|------|------|
| **TPH-YOLOv5** | Transformer Prediction Head 增强小目标 |
| **YOLO-MS** | 多尺度增强小目标检测 |
| **Slicing Aided Hyper Inference (SAHI)** | 切片推理增强小目标 |
| **特征金字塔增强** | BiFPN, PANet 增强多尺度特征 |
| **超分辨率预处理** | ESRGAN + YOLO 两阶段流水线 |

### 3. 旋转目标检测 (Oriented Object Detection)

船舶常呈任意方向，轴对齐框可能包含大量背景。

| 模型 | 说明 |
|------|------|
| **R3Det** | 旋转 RetinaNet |
| **Oriented R-CNN** | 旋转 R-CNN |
| **S2ANet** | 对齐特征旋转检测 |
| **YOLOv8-OBB** | YOLOv8 旋转框版本 |

### 4. 船舶细粒度分类 (Fine-Grained Ship Classification)

在检测框内进一步识别船舶类型：

| 模型 | 说明 |
|------|------|
| **ResNet / EfficientNet** | 船舶类型多分类 |
| **DCL (Destruction & Construction)** | 细粒度分类 |
| **ViT (Vision Transformer)** | 全局特征分类 |

### 5. 域自适应检测 (Domain-Adaptive Detection)

跨数据集/跨海域的泛化：

| 方法 | 说明 |
|------|------|
| **DA-Faster R-CNN** | 图像级 + 实例级对齐 |
| **SFA (Selective Feature Alignment)** | 选择性特征域自适应 |
| **UDA-OVD** | 无监督域自适应目标检测 |

### 6. 半监督/自监督检测

利用未标注数据提升有限标注下的性能：

| 方法 | 说明 |
|------|------|
| **STAC** | 自训练增强一致性 |
| **Unbiased Teacher** | 无偏教师-学生半监督检测 |
| **Dense Teacher** | 密集伪标签半监督 |

## 训练建议

1. **图像尺寸**: 船舶检测建议输入 640×640 或 1280×1280（远距离小目标用大尺寸）
2. **数据增强**: Mosaic、MixUp、Copy-Paste、RandomAffine
3. **Anchor 设计**: 船舶通常为横宽矩形，建议根据数据集聚类重新生成 anchor
4. **评估指标**: mAP@0.5 (COCO), mAP@0.5:0.95；近岸/远海分开评估
5. **类别合并**: 多个数据集合并时注意类别映射统一（如 "boat", "ship", "vessel" → "ship"）

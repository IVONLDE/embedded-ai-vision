# 五类场景测试展示数据集

> 路径: `F:\716数据集\五类场景测试展示数据集`
> 总文件数: ~33,871
> 用途: 五大海事智能场景的综合测试演示

---

## 数据集概览

该数据集覆盖 **5 个核心海事智能化场景**，为 ISG 项目提供端到端的验证能力：

---

## 场景一：多模态数据融合 (Multimodal Data Fusion)

| 属性 | 值 |
|------|-----|
| 路径 | `多模态数据融合/WaterScenes-Mini/` |
| 标注文件 | `information_list.csv`, `train.txt`, `val.txt` |
| 信息字段 | `id, time, lighting, weather, waterway, position` |

### 数据描述
以 **WaterScenes** 数据集为基础的水面场景多模态感知数据，包含图像、雷达点云，标注了光照、天气、水道类型、位置等元信息。

### 可实现的深度学习算法
| 算法方向 | 推荐模型 | 说明 |
|----------|----------|------|
| 多模态目标检测 | MVX-Net, TransFuser | 图像+雷达融合的 3D 目标检测 |
| 多传感器融合 | DeepFusion, BEVFusion | 相机-LiDAR-Radar 统一鸟瞰图感知 |
| 跨模态特征对齐 | CLIP4Perception | 多模态特征空间对齐 |
| 恶劣环境鲁棒感知 | Domain Generalization | 跨天气/光照的分布外泛化 |

---

## 场景二：水下目标检测与识别 (Underwater Object Detection)

| 属性 | 值 |
|------|-----|
| 路径 | `水下目标检测与识别/` |
| 子集 | FLSMDD, NKSID |

### FLSMDD (Forward-Looking Sonar Marine Debris Dataset)
- **图像量**: 2,368 张前视声纳图像
- **类别数**: 10 类水下目标
- **类别**: bottle(449), can(367), chain(226), drink-carton(349), hook(133), propeller(137), shampoo-bottle(99), standing-bottle(65), tire(331), valve(208)

### NKSID (Nankai Sonar Image Dataset)
- **图像量**: 3,023 张声纳图像
- **类别数**: 8 类水下目标
- **类别**: big_propeller, cylinder, fishing_net, floats, iron_pipeline, small_propeller, soft_pipeline, tire

### 可实现的深度学习算法
| 算法方向 | 推荐模型 | 说明 |
|----------|----------|------|
| 声纳图像分类 | ResNet-50, EfficientNet, ViT | 声纳目标多分类 |
| 声纳目标检测 | YOLOv5/v8, Faster R-CNN | 框出声纳图像中的目标 |
| 小样本声纳识别 | Prototypical Networks, MAML | 类别不平衡下的少样本学习 |
| 水下目标语义分割 | U-Net, DeepLabV3+ | 像素级声纳图像分割 |
| 域自适应 | DANN, ADDA | 不同声纳设备间的跨域迁移 |

## 场景三：系统健康状态预估与故障诊断 (System Health & Fault Diagnosis)

| 属性 | 值 |
|------|-----|
| 路径 | `系统健康状态预估与故障诊断/datasets/` |
| 数据格式 | CSV 时序数据 |
| 字段 | `MEAN TEMP` (平均温度), `label` (故障标签) |
| 变体 | 0%~50% 重叠的 6 个滑动窗口版本 |

### 数据描述
内燃机/机械系统的温度传感器时序数据，不同 `overlapping` 比例（0%-50%）对应不同时间窗口的切分策略。

### 可实现的深度学习算法
| 算法方向 | 推荐模型 | 说明 |
|----------|----------|------|
| 故障分类 | 1D-CNN, LSTM, TCN | 时序传感器信号的故障模式识别 |
| 剩余寿命预测 (RUL) | LSTM-RUL, Transformer-RUL | 预测设备离故障还有多长时间 |
| 异常检测 | AutoEncoder, VAE, LSTM-AE | 无监督/半监督异常检测 |
| 健康指标构建 | DeepCORAL, LSTM-HI | 构建连续退化健康指数 |
| 域泛化故障诊断 | DANN, MMD | 不同工况下的跨域故障诊断 |
| 图结构故障诊断 | GCN, GAT | 基于 CSA 图神经网络的故障推理 |

## 场景四：智能决策与指挥控制 (Intelligent Decision & Command)

| 属性 | 值 |
|------|-----|
| 路径 | `智能决策与指挥控制/data/` |
| 原始数据 | `raw/ship_data.csv` (AIS 船舶数据), `raw_ais_full/zenodo_raw_filtered.csv` |
| 预处理 | `processed/preprocessor.pkl` |
| 数据字段 | `longitude, latitude, speed, course` |

### 数据描述
大规模 AIS 船舶航行记录，包含船舶的经纬度、航速、航向等动态信息，可用于态势感知和决策推理。

### 可实现的深度学习算法
| 算法方向 | 推荐模型 | 说明 |
|----------|----------|------|
| 航迹预测 | LSTM, Transformer, TrajGRU | 短期/长期船舶航迹预测 |
| 碰撞风险评估 | GNN + RiskNet | 多船交互场景碰撞概率预测 |
| 海上交通流预测 | ST-GCN, GraphWaveNet | 海域交通密度时空预测 |
| 强化学习避障 | DQN, PPO | 基于深度强化学习的自主避障决策 |
| 路径规划 | A* + RL, Motion Transformer | 智能航线规划与推荐 |

## 场景五：舰船目标识别与跟踪 (Ship Recognition & Tracking)

| 属性 | 值 |
|------|-----|
| 路径 | `舰船目标识别与跟踪/Ship_Detection_v5/` |
| 格式 | YOLOv5 目标检测格式 |
| 类别数 | 1 (ship) |
| 数据划分 | `train/images/` + `train/labels/`, `valid/images/` + `valid/labels/` |
| 来源 | Roboflow (CC BY 4.0) |

### 数据描述
标注好的舰船目标检测数据集，YOLO 格式标注文件（`.txt`），可用于训练和评估舰船检测模型。

### 可实现的深度学习算法
| 算法方向 | 推荐模型 | 说明 |
|----------|----------|------|
| 舰船目标检测 | YOLOv5/v8/v10, DETR, RT-DETR | 实时/高精度舰船检测 |
| 多目标跟踪 (MOT) | DeepSORT, ByteTrack, BoT-SORT | 视频流中的舰船持续跟踪 |
| 舰船重识别 (ReID) | OSNet, TransReID | 跨摄像头舰船身份重识别 |
| 旋转框检测 | R3Det, Oriented R-CNN | 任意方向舰船的旋转框检测 |
| 小目标舰船检测 | TPH-YOLOv5, YOLO-MS | 远距离小舰船目标增强检测 |

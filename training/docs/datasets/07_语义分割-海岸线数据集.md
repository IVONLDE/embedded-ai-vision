## 数据描述

海陆分割是海事遥感的经典任务，目的是对卫星/航拍图像中的每一个像素进行分类，判断其属于**海洋**还是**陆地**。该数据集提供像素级标注，适合训练语义分割模型。

### 典型场景

- 海岸线提取与变化监测
- 港口/近海区域自动制图
- 船舶航行可通行区域分析
- 沿海环境保护与监测
- 岛屿/礁石自动检测

## 可实现的深度学习算法

### 1. 经典语义分割

| 模型 | 参数量 | 说明 |
|------|--------|------|
| **U-Net** | ~31M | 经典编码器-解码器，对小数据集友好 |
| **U-Net++** | ~36M | 嵌套跳跃连接，更细粒度特征融合 |
| **Attention U-Net** | ~35M | 注意力门控聚焦目标区域 |
| **DeepLabV3+** | ~41M | 空洞卷积 + 空间金字塔池化 |
| **PSPNet** | ~47M | 金字塔场景解析网络 |

### 2. 轻量级实时分割 (边缘部署)

| 模型 | 说明 |
|------|------|
| **BiSeNet / BiSeNetV2** | 双路网络（空间路径 + 语义路径） |
| **STDC-Seg** | 超轻量实时分割网络 |
| **PIDNet** | 比例积分微分双分支网络 |
| **MobileNetV3 + DeepLab** | 移动端轻量骨干 |

### 3. Transformer 分割

| 模型 | 说明 |
|------|------|
| **SegFormer** | 层级 Transformer + MLP 解码器 |
| **Mask2Former** | 掩码分类统一分割 |
| **Swin-UNETR** | Swin Transformer + U-Net |
| **SETR** | 纯 Transformer 编码器分割 |

### 4. 边缘精细化

海陆边界往往是细长不规则的，需要特别的边缘增强策略：

| 方法 | 说明 |
|------|------|
| **边缘感知损失 (Boundary Loss)** | 直接优化边界像素 |
| **CRF 后处理 (Dense CRF)** | 条件随机场边缘精修 |
| **边缘分支多任务** | 分割头 + 边缘检测头联合训练 |
| **Shape-Aware Loss** | 形状感知损失函数 |

### 5. 半监督/弱监督分割

减少标注成本（像素级标注非常昂贵）：

| 方法 | 说明 |
|------|------|
| **Mean Teacher** | 教师-学生一致性正则化 |
| **FixMatch** | 伪标签 + 强弱增强 |
| **Class Activation Maps (CAM)** | 仅用图像级标签的弱监督 |

### 6. 海岸线变化检测 (Change Detection)

| 模型 | 说明 |
|------|------|
| **Siamese U-Net** | 孪生网络比较两时相分割差异 |
| **ChangeFormer** | Transformer 变化检测 |
| **BIT (Bitemporal Image Transformer)** | 双时相 Transformer |

## 训练建议

1. **输入尺寸**: 遥感图像通常较大，建议切块 (tile) 训练，如 256×256 或 512×512 的滑动窗口
2. **类别不平衡**: 纯海/纯陆的图块可能占多数，需过滤或加权
3. **数据增强**: 随机旋转、翻转、色彩抖动、CLAHE 增强对比度
4. **损失函数**: Dice Loss + BCE Loss 组合，或 Tversky Loss（处理不平衡）
5. **评估指标**: mIoU, Dice Coefficient, Precision/Recall per-class
6. **后处理**: 可选的形态学操作（膨胀+腐蚀）或 CRF 精修边界

# 故障诊断数据集 (Fault Diagnosis Dataset)

> 路径: `F:\716数据集\故障诊断数据集`
> 模态: 传感器时序信号 / 图结构数据

## 数据集概览

包含内燃机/机械系统的故障诊断基准数据，以及图结构因果分析方法论数据。

## 子集一：内燃机故障诊断基准数据集

| 属性 | 值 |
|------|-----|
| 路径 | `内燃机故障诊断基准数据集/` |
| 训练集 | `trainingdata.zip` |
| 测试集 | `testdata.zip` |
| 数据格式 | 时序传感器 CSV |
| 特征字段 | `MEAN TEMP` (平均温度) |

### 数据描述

内燃机在不同故障模式下的传感器监测数据。数据以滑动窗口切分，提供 6 种不同的窗口重叠比例（0% ~ 50%）。

- `0_percent_overlapping.csv` → 无重叠，窗口不共享数据
- `10_percent_overlapping.csv` → 10% 重叠
- `20_percent_overlapping.csv`
- `30_percent_overlapping.csv`
- `40_percent_overlapping.csv`
- `50_percent_overlapping.csv` → 半重叠窗口

不同重叠率影响训练样本量和时序连续性，适合研究**窗口策略对时序模型性能的影响**。

### 可实现的深度学习算法

#### 1. 时序故障分类

| 模型 | 说明 |
|------|------|
| **1D-CNN** | 一维卷积提取局部时序模式 |
| **LSTM / BiLSTM** | 长短时记忆网络捕获长期依赖 |
| **TCN (Temporal ConvNet)** | 空洞因果卷积，训练更快 |
| **InceptionTime** | 多尺度时序卷积集成 |
| **Transformer Encoder** | 自注意力全局时序建模 |

#### 2. 迁移学习故障诊断

| 方法 | 说明 |
|------|------|
| **Fine-tune pre-trained model** | 源工况预训练 + 目标工况微调 |
| **DANN (Domain Adversarial NN)** | 对抗域自适应 |
| **MCD (Maximum Classifier Discrepancy)** | 无监督域自适应 |
| **DeepCORAL** | 二阶统计量对齐 |

#### 3. 剩余使用寿命预测 (RUL)

| 模型 | 说明 |
|------|------|
| **LSTM-RUL** | LSTM 编码退化趋势 |
| **CNN-LSTM** | CNN 提取局部特征 + LSTM 时序建模 |
| **Transformer-RUL** | 注意力机制的退化预测 |

#### 4. 无监督异常检测

| 模型 | 说明 |
|------|------|
| **AutoEncoder (AE)** | 重构误差异常判据 |
| **VAE (Variational AE)** | 概率重构异常检测 |
| **LSTM-AE** | 时序自编码器异常检测 |
| **DeepSVDD** | 深度单类分类 |

## 子集二：Graph-Structured CSA (图结构因果分析)

| 属性 | 值 |
|------|-----|
| 路径 | `Graph-Structured-about-CSA-main.zip` |
| 格式 | 代码 + 数据 (压缩包) |
| 方法论 | 图因果结构分析 |

### 数据描述

基于**图神经网络 (GNN)** 的因果结构分析方法，将故障诊断转化为图中的因果推理问题。传感器之间不是孤立的，而是存在因果依赖关系。

### 可实现的深度学习算法

| 模型 | 说明 |
|------|------|
| **GCN (Graph Convolutional Network)** | 图卷积故障特征提取 |
| **GAT (Graph Attention Network)** | 注意力加权的异构图故障诊断 |
| **GraphSAGE** | 归纳式图节点状态分类 |
| **CausalGNN** | 因果图结构学习 + 故障推理 |
| **GIN (Graph Isomorphism Net)** | 图同构网络故障分类 |

## 训练建议

1. **归一化**: 传感器信号务必做标准化 (Z-score) 或 MinMax 归一化
2. **窗口策略**: 初始实验建议用 30%~50% 重叠窗口，平衡样本量与时间连续性
3. **评估指标**: 不平衡数据用 Macro F1、AUC-ROC；RUL 预测用 RMSE、Score
4. **过采样**: 如果故障样本稀少，考虑 SMOTE 或时间序列增强 (jittering, scaling, time-warping)
5. **图构建**: GNN 方法需根据传感器物理拓扑或相关性先构建邻接矩阵
